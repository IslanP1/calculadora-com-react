# Desafio calculadora da rocketseat

Essa desafio trata-se de um projeto simples que realizei para aperfeiçoamento das minha habilidades com react.

![Calculadora](./public/calculadora.png)

## Funcionalidades:
- Soma
- Subtração
- Divisão
- Multiplicação
- Porcentagem
